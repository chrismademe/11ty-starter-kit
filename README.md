# 11ty Starter Kit

## What's included?

-   Sass compilation out of the box
-   Starter CSS
-   `.htaccess`, `robots.txt` and `sitemap.xml` ready to go
-   Custom 404 page
-   Eleventy Navigation plugin

## Roadmap

-   Create a skeleton site to get started with

## Get Started

```bash
npx degit chrismademe/11ty-starter-kit
cd 11ty-starter-kit
npm install
npm run dev
```
