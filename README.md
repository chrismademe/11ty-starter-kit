# 11ty Starter Kit

## What's included?

-   Boilerplate HTML (responsive meta tag, favicons etc.)
-   Starter CSS
-   Responsive typescale using `clamp`
-   SEO ready
-   Custom 404 page
-   Eleventy Navigation plugin

## Roadmap

-   Create a skeleton site to get started with

## Get Started

```bash
npx degit chrismademe/11ty-starter-kit
cd 11ty-starter-kit
npm install
npm run dev
```
